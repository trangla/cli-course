## Data Types

- Number
```js
10;     // integer
12.45   // float / double
-55666  // 
```

- String (Combination of Characters/letter)
```js
"Syed";     // double quotes
'Zakeer';   // single quotes
`Hussain`;  // backtick 

Harish; // not a string, will treat as reference (variable)
```

- Boolean (Yes/No or ON/OFF)
```js
true;   // YES
false;  // NO
```
