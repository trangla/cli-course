# Programming Basics

Here we use **Javascript** language

First enter into NodeJS REPL
```bash
$ > node
``` 

Then you will see like below
```bash
Welcome to Node.js v20.12.1.
Type ".help" for more information.
> 
```


Now do basic Math operations

### Addition
```js
> 10 + 20
```
Output: `30`

### Subtraction
```js
> 10 - 20
```
Output: -10
